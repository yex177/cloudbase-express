# 用户手册

## 1. 系统概述

微信小程序后台管理系统是一个功能完善、性能稳定的PC端管理系统，用于管理微信小程序的产品、媒体、订单和物流等业务。

### 1.1 系统功能

- 产品管理：产品的添加、编辑、删除、上下架和库存管理
- 媒体管理：图片的上传、预览、编辑和删除
- 数据统计：销售报表、数据可视化和导出功能
- 物流管理：物流公司配置和物流状态查询

### 1.2 系统要求

- 浏览器：Chrome、Firefox、Safari、Edge等现代浏览器
- 分辨率：建议1366x768及以上
- 网络：稳定的网络连接

## 2. 登录系统

1. **打开登录页面**
   在浏览器中输入系统地址，如 `http://localhost:8000`

2. **输入登录信息**
   - 用户名：系统管理员分配的用户名
   - 密码：系统管理员分配的密码

3. **点击登录按钮**
   登录成功后，系统会自动跳转到仪表盘页面

## 3. 仪表盘

仪表盘是系统的首页，显示了系统的关键指标和数据概览。

### 3.1 关键指标

- 总销售额：系统的总销售金额
- 总订单数：系统的总订单数量
- 产品数量：系统的产品总数
- 库存预警：低库存产品的数量

### 3.2 销售趋势

显示销售数据的趋势图表，支持按日、周、月查看。

### 3.3 热销产品

显示销售数量最多的前10个产品。

## 4. 产品管理

### 4.1 产品列表

- **搜索产品**：在搜索框中输入产品名称，点击搜索按钮
- **筛选产品**：通过分类和状态下拉框筛选产品
- **分页查看**：通过分页控件查看更多产品

### 4.2 添加产品

1. 点击"添加产品"按钮
2. 填写产品信息：
   - 产品名称：必填
   - 产品描述：可选
   - 价格：必填
   - 库存：必填
   - 库存预警：可选，默认10
   - 分类：必填
   - 标签：可选
   - 状态：必填，默认上架
   - 图片：可选，最多上传5张
3. 点击"确定"按钮

### 4.3 编辑产品

1. 在产品列表中找到要编辑的产品
2. 点击"编辑"按钮
3. 修改产品信息
4. 点击"确定"按钮

### 4.4 删除产品

1. 在产品列表中找到要删除的产品
2. 点击"删除"按钮
3. 在弹出的确认对话框中点击"确定"按钮

### 4.5 上下架产品

1. 在产品列表中找到要上下架的产品
2. 点击"上架"或"下架"按钮
3. 系统会自动更新产品状态

### 4.6 批量操作

1. 在产品列表中勾选多个产品
2. 点击批量操作按钮：
   - 批量上架：将选中的产品上架
   - 批量下架：将选中的产品下架
   - 批量删除：删除选中的产品

### 4.7 库存预警

系统会自动检测库存低于预警值的产品，并在仪表盘和库存预警页面显示。

## 5. 媒体管理

### 5.1 上传图片

1. 点击"点击上传图片"区域
2. 选择要上传的图片（最多5张）
3. 系统会自动上传图片并显示在图片列表中

### 5.2 查看图片

图片上传后会显示在图片列表中，鼠标悬停在图片上可以预览。

### 5.3 删除图片

1. 在图片列表中找到要删除的图片
2. 点击图片上的删除按钮
3. 系统会自动删除图片

## 6. 数据统计

### 6.1 销售统计

1. 选择日期范围
2. 选择时间间隔（按天、按周、按月）
3. 点击"查询"按钮
4. 系统会显示销售趋势图表和客单价

### 6.2 热销产品

系统会显示销售数量最多的前10个产品。

### 6.3 导出报表

1. 选择日期范围
2. 点击"导出Excel"或"导出PDF"按钮
3. 系统会下载相应格式的报表

### 6.4 库存统计

系统会显示产品总数、低库存产品数和缺货产品数。

## 7. 物流管理

### 7.1 物流公司管理

#### 7.1.1 添加物流公司

1. 点击"添加物流公司"按钮
2. 填写物流公司信息：
   - 公司名称：必填
   - 公司代码：必填
   - API地址：必填
   - API密钥：必填
   - 状态：默认启用
3. 点击"确定"按钮

#### 7.1.2 编辑物流公司

1. 在物流公司列表中找到要编辑的公司
2. 点击"编辑"按钮
3. 修改物流公司信息
4. 点击"确定"按钮

#### 7.1.3 删除物流公司

1. 在物流公司列表中找到要删除的公司
2. 点击"删除"按钮
3. 在弹出的确认对话框中点击"确定"按钮

### 7.2 订单物流管理

#### 7.2.1 更新物流信息

1. 在订单列表中找到要更新的订单
2. 点击"更新物流"按钮
3. 选择物流公司和填写物流单号
4. 点击"确定"按钮

#### 7.2.2 查询物流状态

1. 在订单列表中找到要查询的订单
2. 点击"查询状态"按钮
3. 系统会显示物流状态和物流轨迹

## 8. 系统设置

### 8.1 用户管理

- 添加用户：创建新的系统用户
- 编辑用户：修改用户信息和权限
- 删除用户：删除系统用户

### 8.2 权限管理

- 角色管理：创建和管理用户角色
- 权限分配：为角色分配权限

### 8.3 系统配置

- 基本设置：配置系统名称、Logo等
- 邮件设置：配置邮件服务器
- 短信设置：配置短信服务

## 9. 常见问题

### 9.1 登录失败

- 检查用户名和密码是否正确
- 检查网络连接是否正常
- 联系系统管理员重置密码

### 9.2 上传图片失败

- 检查图片大小是否超过限制（10MB）
- 检查图片格式是否支持（jpg、jpeg、png、gif）
- 检查网络连接是否正常

### 9.3 物流查询失败

- 检查物流单号是否正确
- 检查物流公司API配置是否正确
- 检查网络连接是否正常

### 9.4 数据统计不准确

- 检查日期范围是否正确
- 检查数据是否已同步
- 联系系统管理员检查数据
